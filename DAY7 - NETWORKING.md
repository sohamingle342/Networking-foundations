## Day 7: Network Evolution - Convergence POE and Quality of Service (QoS)

### 🧠 Core Concepts Learned 
* **What is Convergence?** In the past, companies had to build completely separate wiring systems ("silos") for their computers, telephone lines (VoIP), and security systems. *Convergence* is the process of bringing all these separate technologies together so they run on standard computer network cables (like Cat5/Cat6) and use standard IP protocols.
* **Power over Ethernet (PoE):** A technology that allows a single network cable to send both internet data *and* electrical power to a device at the same time. This is used to power things like IP security cameras, VoIP desk phones, and wireless access points without needing to plug them into an electrical wall outlet.
* **Parallel Networks:** When certain data traffic (like voice calls or security video streams) is so heavy or sensitive that it threatens to slow down normal computer traffic, engineers will split them onto their own dedicated, isolated network pathways to prevent lag and performance drops.
* **Intelligence at the Edge:** Modern network devices are designed with their own computing power so they can process data directly on the device itself (at the "edge" of the network) instead of constantly sending raw, unorganized data back to a main central server, saving massive amounts of network bandwidth.
