## Day 5:  - SOHO Routers & Gateway Operations

### 🧠 Core Concepts Learned 
* **SOHO Architecture:** Understood that a Small Office/Home Office (SOHO) router is a multi-functional hybrid device combining a router, a switch, a firewall, and a wireless access point into a single physical unit.
* **DHCP Server & Scope Management:** The router acts as a Dynamic Host Configuration Protocol (DHCP) server, automatically assigning temporary IP addresses to internal devices from a designated pool (Scope). Proper administration requires excluding static IPs assigned to critical infrastructure (like local servers).
* **Port Forwarding:** Explicitly directing incoming traffic from the public internet hitting a specific port on the router directly to a specific internal private IP address on the local network. 

* **Basic Firewall Rules:** The SOHO router provides the initial boundary defense by blocking unrequested inbound ports, preventing external unauthorized scans from hitting internal hosts.
